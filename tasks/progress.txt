### API 端点
| 端点 | 方法 | 描述 |
|------|------|------|
| /api/records | POST | 创建记录 |
| /api/records | GET | 获取记录列表（分页、筛选、搜索） |
| /api/records/{id} | GET | 获取记录详情 |
| /api/records/{id} | PUT | 更新记录 |
| /api/records/{id} | DELETE | 删除记录 |
| /api/records/classify | POST | AI 内容分类 |
| /api/records/transcribe | POST | 语音转文字 |
| /api/records/upload/audio | POST | 上传音频文件 |
| /api/records/stats/summary | GET | 获取记录统计 |

## Sprint 3 进度 ✅ 已完成

### 已完成任务
- [x] US-004: AI 建议生成
  - Suggestion 数据模型（建议内容、维度、用户反馈、创建时间）
  - 建议相关 Schema 定义
  - 建议 API 端点：
    - POST /api/suggestions/generate - 生成今日建议
    - GET /api/suggestions/today - 获取今日建议
    - GET /api/suggestions/history - 获取建议历史
    - POST /api/suggestions/{id}/feedback - 标记建议有用/无用
    - GET /api/suggestions/{id} - 获取单条建议
  - 扩展 AI 服务：
    - 分析用户历史记录（按维度统计、趋势分析）
    - 生成个性化建议（调用 LLM API，支持模拟数据）
  - 前端建议页面（三个维度分组展示、反馈按钮、历史弹窗）
  - 首页集成（显示建议摘要、快捷入口）
