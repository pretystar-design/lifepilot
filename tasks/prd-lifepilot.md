## 1. 产品概述

### 1.1 产品定位
LifePilot 是一款个人生活智能助手，通过记录用户日常生活数据，利用 AI 深度分析，在学习成长、理财规划、健康管理三个维度提供个性化建议。

### 1.2 核心价值
- **极简记录**：支持文字、语音、视频多种输入方式
- **AI 洞察**：不是简单统计，而是深度分析生活模式
- **三维度建议**：学习成长、理财规划、健康三管齐下
- **持续进化**：记录越多，建议越精准

### 1.3 目标用户
- 职场新人（22-28岁）：需要理财启蒙、健康管理、学习规划
- 自我提升者（28-35岁）：优化理财、关注健康、持续学习
- 中年规划者（35-45岁）：家庭财务规划、健康风险管理

### 3.3 数据库设计

#### 用户表 (users)
```sql
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    phone VARCHAR(20) UNIQUE,
    nickname VARCHAR(50),
    avatar_url VARCHAR(500),
    openid VARCHAR(100) UNIQUE,  -- 微信 openid
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);
```

#### 记录表 (records)
```sql
CREATE TABLE records (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    title VARCHAR(200),
    content TEXT NOT NULL,
    category VARCHAR(20) NOT NULL,  -- learning/finance/health
    tags TEXT[],
    audio_url VARCHAR(500),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);
```

#### AI建议表 (suggestions)
```sql
CREATE TABLE suggestions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    date DATE NOT NULL,
    category VARCHAR(20) NOT NULL,
    content TEXT NOT NULL,
    is_helpful BOOLEAN,
    created_at TIMESTAMP DEFAULT NOW()
);
```

#### 用户设置表 (user_settings)
```sql
CREATE TABLE user_settings (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id) UNIQUE,
    learning_enabled BOOLEAN DEFAULT TRUE,
    finance_enabled BOOLEAN DEFAULT TRUE,
    health_enabled BOOLEAN DEFAULT TRUE,
    learning_goal TEXT,
    finance_goal TEXT,
    health_goal TEXT,
    reminder_enabled BOOLEAN DEFAULT TRUE,
    reminder_time TIME DEFAULT '21:00:00',
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);
```

## 附录

### A. 竞品分析
| 产品 | 优势 | 劣势 |
|------|------|------|
| 小红书 | 内容丰富、社区活跃 | 无 AI 建议、非个人成长导向 |
| 记账软件 | 理财功能强 | 单维度、无 AI 分析 |
| 健康类 App | 数据全面 | 单维度、无生活整合 |
| Notion | 灵活强大 | 学习成本高、无 AI 建议 |

### B. 差异化优势
1. **三维度整合**：学习/理财/健康一站式
2. **AI 深度分析**：不只是记录，更是洞察
3. **极简体验**：30秒完成记录
4. **持续进化**：越用越懂你
